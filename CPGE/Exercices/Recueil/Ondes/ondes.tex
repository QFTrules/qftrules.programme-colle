\begin{exo}[PCSI][1]{Interférence d’ondes radio}
﻿Deux sources $S_1$ et $S_2$ en phase génèrent des ondes radio sinusoïdales de longueur d'onde $\lambda=\SI{50}{cm}$ et de même amplitude $A$. $S_1$ est placée à l'origine de l'axe $x$ et $S_2$ peut être placée n'importe où sur le demi axe ($Ox$), à une distance $d<\SI{1}{m}$ de $S_1$.

\begin{quest}
\QR{
Réalisez un schéma clair et détaillé de la situation.
}{}

\QR{
Déterminez à quel(s) endroit(s) placer $S_2$ pour qu'il y ait des interférences constructives en $x=\SI{15}{cm}$.
}{
La condition pour avoir des interférences constructives est :
\eq{
	x - |x_2-x| = n\lambda\quad n \in \mathbb{Z}
	}
Si la source 2 est à gauche de $x$, cette relation devient :
\eq{
	x_2 = n\lambda,
	}
ce qui n'a pas de solution avec les données de l'exercice. Si la source 2 est à droite de $x$, alors la condition devient :
\eq{
	\boxed{x_2 = 2x + n\lambda}
}
L'application numérique donne
\eq{
	\boxed{x_2 = \SI{30}{\centi\metre}}
	\quad\text{et}\quad
	\boxed{x_2 = \SI{80}{\centi\metre}}.
	}

}

\QR{
Même question pour des interférences destructives en $x=\SI{15}{cm}$.
}{
La condition pour avoir des interférences destructives est :
\eq{
	x - |x_2-x| = \left( n + \frac{1}{2} \right) \lambda\quad n \in \mathbb{Z}
	}
Si la source 2 est à gauche de $x$, cette relation devient :
\eq{
	x_2 = \left( n + \frac{1}{2} \right)\lambda,
	}
ce qui n'a pas de solution avec les données de l'exercice. Si la source 2 est à droite de $x$, alors la condition devient :
\eq{
	\boxed{x_2 = 2x + \left(n+\frac{1}{2}\right)\lambda}
}

L'application numérique donne
\eq{
	\boxed{x_2 = \SI{5}{\centi\metre}}
	\quad\text{et}\quad
	\boxed{x_2 = \SI{55}{\centi\metre}}.
	}

}

\QR{
On suppose que la seconde source est à une distance $l$ de $\SI{70}{cm}$ de la première. Détermine l'interfrange, c'est à dire la distance entre deux points successifs dans le même état intérférentiels.
}{
L'onde reçue en $x>x_2$ est :
\eq{
s(x,t) = A\cos(\omega t - kx) + A\cos (\omega t - k(x-x_2))
	}
En utilisant des formules trigonométriques
\eq{
\cos (p) + \cos (q) = 2\cos \left(\frac{p+q}{2} \right)\cos \left( \frac{p-q}{2}\right),
}
on trouve que l'onde reçu en $x$ est :
\eq{
s(x,t) = A\left[ \cos(\omega t - kx + kx_2/2) \cos(kx_2/2) \right].
	}
L'interfrange est alors :
\eq{
i=l_1-l_2
\quad\text{tel que} \quad
s(l_1,t) = s(l_2,t)
	}
On trouve alors :
\eq{
kl_1 = kl_2+2\pi
\quad \Rightarrow\quad
\boxed{i = \frac{2\pi}{k} = \lambda}.
	}
  }
\end{quest}

\end{exo}

\begin{exo}[PCSI][2]{Transmission entre deux cordes}
  Une corde infinie est constituée de deux parties :
	\begin{itemize}
	\item[$\bullet$] $x<0$ :  masse linéique $\mu_1$, tension $T$
	\item[$\bullet$] $x>0$ :  masse linéique $\mu_2$, même tension $T$
	\end{itemize}
	Une onde progressive se dirige vers le point $O$ en provenant de la région des $x$ négatifs. On notera $c_1$ (respectivement $c_2$) la célérité des ondes susceptibles de se propager sur la partie $x<0$ (respectivement $x>0$) de la corde.\\

	On note $y_i(x,t)$, $y_r(x,t)$, $y_t(x,t)$ les élongations correspondant à l'onde incidente, réfléchie et transmise. \`A ces élongations correspondent les ondes de vitesse respectives
	\eq{
		v_i(x,t)=\diffp{y_i(x,t)}{ t} \quad ,\quad
		v_r(x,t)=\diffp{y_r(x,t)}{t} \quad ,\quad
		v_t(x,t)=\diffp{ y_t(x,t)}{ t}.
	}
\begin{quest}

  \QR{
  	Définir une onde progressive se propageant à la vitesse $v$ selon les $x$ croissants. Justifier que $\cfrac{\partial y_i(x,t)}{\partial x}=-\cfrac{v_i(x,t)}{c_1}$.
  }{
  	% Comme
  	% $
  	% 	y_i(x,t)=f(t-x/c_1)\qet y_r(x,t)=g(t+x/c_1)\qet y_t=h(t-x/c_2)
  	% $

  	% On en déduit
  	% $
  	% 	\diffp{y_i(x,t)}{x}=\diff{f(u)}{t}\times \diffp{t}{u}\times \diffp{u}{x}\qavec u=t-x/c_1
  	% $

  	% On trouve alors
  	% $
  	% 	\diffp{y_i(x,t)}{x}=\diff{f(t-x/c)}{t}\times\pa{-\cfrac{1}{c_1}}
  	% $
  	% $
  	% 	\qsoit
  	% 	\cfrac{\partial y_i(x,t)}{\partial x}=-\cfrac{v_i(x,t)}{c_1}
  	% $
  }

  \QR{
  	Déterminer les coefficients de réflexion $r$ et de transmission $t$ relatifs à la vitesse. On introduira la quantité $\eta=\sqrt{\mu_2/\mu_1}$.

  	Que se passe-t-il pour les cas limites  $\mu_2\rightarrow \infty$ et $\mu_1=\mu_2$~?
  }{
  	On écrit la continuité de la corde en $y=0$ et en dérivant par rapport au temps, on a
  	\eq{
  		\boxed{v_i(0,t)+v_r(0,t)=v_t(0,t)}
  	 }

  	 On écrit la continuité de la tangente à la corde en $y=0$ :
  	 \eq{
  		 \diffp{y_i(x,t)}{x}(x=0) +\diffp{y_r(x,t)}{x}(x=0)= \diffp{y_t(x,t)}{x}(x=0)
  		}
  	\eq{
  	 	\qsoit
  	 	\boxed{-\cfrac{v_i(0,t)}{c_1}+\cfrac{v_r(0,t)}{c_1}=-\cfrac{v_t(0,t)}{c_2}}
  	 }

  	 En exploitant ces deux relation  et avec $\eta=c_1/c_2$, on a
  	 \eq{
  		\boxed{r=\cfrac{1-\eta}{1+\eta}\qet t=1+r=\cfrac{2}{1+\eta}}
  	  }

  	  Cas limites :
  	  \begin{itemize}
  	\item $\eta\Rightarrow +\infty$ : $r=-1$, $ t=0$ l'onde se "heurte" à un mur, elle est uniquement réfléchie
  	\item $\eta=1$ : $r=0$, $t=1$ : il n'y a pas de changement de milieu, donc pas de réflexion
  	  \end{itemize}
  }

  \QR{
  	Définir la puissance $\Pi (x,t)$ transférée en $x$ dans le sens des $x>0$ dans le cas général d'une corde infinie de tension $T$ et de masse linéique $\mu$. On exprimera $\Pi (x,t)$ en fonction de $T$, $c$, la célérité de l'onde, et $v(x,t)$ sa vitesse transversale. On se placera dans le cas d'une onde progressive.
  }{
  	\eq{
  		\Pi(x,t)=-\vec{T}(x,t)\cdot\vec{v}(x,t)=-T_y(x,t)\times\diffp{y(x,t)}{t}
  	}

  	Or $T_y(x,t)=T\sin(\alpha)\approx T\alpha=T\diffp{y(x,t)}{x}$ et pour une onde progressive $\diffp{y(x,t)}{x}=-\cfrac{v(x,t)}{c}$ on en déduit
  	\eq{
  		\boxed{\Pi(x,t)=\cfrac{T}{c}v^2(x,t)}
  	}
  }

  \QR{
  	En traduisant la conservation de l'énergie en $x=0$, établir une relation entre $r$, $t$ et $\eta$. Vérifier que cette expression est compatible avec les expressions obtenues précédemment.
  }{
  	\eq{
  		\Pi(0^-,t)=-\cfrac{T_y(0^-,t)}{c_1}v(0^-,t)
  	}

  	\eq{
  		T_y(0^-,t)=T\pa{\diffp{y_i}{x}(x=0)+\diffp{y_r}{x}(x=0)}=\cfrac{T}{c_1}\pa{v_r(0,t)-v_i(0,t)}
  	}

  	Comme $v(0^-,t)=v_r(0,t)+v_i(0,t)$, on en déduit
  	\eq{
  		\Pi(0^-,t)=\cfrac{T}{c_1}\pa{v_i(0,t)-v_r(0,t)}\times \pa{v_i(0,t)+v_r(0,t)}
  	}
  	\eq{
  		\qsoit
  		\boxed{\Pi(0^-,t)=\cfrac{T}{c_1}\pa{v_i^2(0,t)-v_r^2(0,t)}}
  	}

  	Pour $x>0$ l'onde est progressive, donc
  	\eq{
  		\boxed{\Pi(0^+,t)=\cfrac{T}{c_2}v_t^2(0,t)}
  	}

  	La continuité de la puissance donne
  	\eq{
  		\cfrac{T}{c_1}\pa{v_i^2(0,t)-v_r^2(0,t)}=\cfrac{T}{c_2}v_t^2(0,t)
  	}
  	\eq{
  		\qsoit
  		v_i^2(0,t)-v_r^2(0,t)=\cfrac{c_1}{c_2}v_t^2(0,t)
  	}

  	On en déduit $\boxed{1-r^2=\eta t^2}$.

  	On vérifie qu'avec les expressions de $r$ et $t$ établies précédemment; cette relation est vérifiée.

  }
\end{quest}
\end{exo}

\begin{exo}[PCSI][1][colle]{Autour de la corde de Melde}
	\Notions{\item Physique des ondes \task Corde vibrante \task Onde stationnaire \task Oscillateur harmonique}
Lors d'une manipulation avec la corde de Melde (régime oscillant forcé), on obtient les résultats suivants pour une longueur $L=\SI{117}{cm}$ de corde et une masse $M=\SI{25}{g}$ attachée à l'extrémité non reliée au vibreur pour tendre la corde.
\begin{quest}
\item On observe deux fuseaux pour une fréquence de résonance de \SI{18}{Hz} et trois fuseaux pour une fréquence de résonance de \SI{27}{Hz}. Ces valeurs numériques sont-elles compatibles entre elles ? Quelles seraient les fréquences de résonances suivantes ?
\sol{
	Les fréquences de résonance observées correspondent à des modes de vibration de la corde. Pour une corde fixée à ses deux extrémités, les fréquences de résonance sont données par la relation :
	$$
	f_n = n \frac{c}{2L},
	$$
	où $n$ est un entier positif, et $c$ est la vitesse de propagation de l'onde dans la corde.

	Pour le mode $n=2$ (deux fuseaux), on a :
	$$
	f_2 = 2 \frac{c}{2L} = \frac{c}{L} = \SI{18}{Hz}.
	$$
	Pour le mode $n=3$ (trois fuseaux), on a :
	$$
	f_3 = 3 \frac{c}{2L} = \frac{3c}{2L} = \SI{27}{Hz}.
	$$
	Les valeurs numériques sont donc compatibles entre elles, puisqu'on a la relation 
	\eqbox{
	\frac{f_3}{f_2} = \frac{27}{18} = \frac{3}{2}.
	}
}
\item Déterminer la valeur de la tension de la corde, puis en déduire un ordre de grandeur de la masse linéique de la corde $\mu$ sachant que la vitesse des ondes le long de la corde est donnée par $c=\sqrt{T/\mu}$. Comment pourrait-on vérifier ce dernier résultat expérimentalement ?
\sol{
	La tension dans la corde est due au poids de la masse suspendue, donc :
	% $$
	\eqbox{
		T = Mg = \SI{25}{g} \times \SI{9,81}{m.s^{-2}} = \SI{0,245}{N}.
		}
	% $$
	La vitesse des ondes dans la corde est donnée par :
	% $$
	\eqbox{
		c = \sqrt{\frac{T}{\mu}}.
		}
	% $$
	On peut donc en déduire la masse linéique de la corde en fonction des données du problème,
	% $$
	\eqbox{
		\mu = \frac{T}{c^2} \simeq  \frac{T}{f_2^2L^2 } = \SI{5.5e-4}{kg.m^{-1}}.
		}
	Pour vérifier ce résultat expérimentalement, on pourrait mesurer la fréquence de résonance de la corde pour différentes tensions et en déduire la masse linéique à partir de la relation $c = \sqrt{T/\mu}$.
}
\end{quest}
\end{exo}

\begin{exo}[MPSI]{Interférences d'ondes à la surface de l'eau}

Deux cailloux identiques tombent dans l'eau à une distance $a$ l'un de l'autre, générant ainsi deux ondes circulaires à la surface de l'eau. La première source se trouve à l'origine $O$ d'un repère $(x,y)$ décrivant la surface de l'eau, l'autre se trouvant en $x=a$. On suppose que la hauteur $z$ de la perturbation à la surface de l'eau induite par la source a pour expression
$$
z(r,t) = z_0(r-ct)\cos[2\pi(r-ct)/\lambda]
$$
où $z_0$ donne la forme de l'enveloppe de l'onde, $c$ est la célérité de l'onde, $\lambda$ la longeur d'onde, et $r$ la distance à la source.


\begin{quest}
\item S'agit-il d'ondes progressives ? On suppose que la célérité $c$ d'une telle onde dépend de l'intensité de la pesanteur $g$ et de la longueur d'onde $\lambda$. Trouver son expression par analyse dimensionnelle.

\sol{
oui car forme $f(r-ct)$.
}


\item Déterminer la différence de marche $\delta(r,\theta)$ entre les deux ondes en un point de la surface de coordonnée polaire $(r,\theta)$ dans la limite $r \gg a$. Déterminer le lieu des interférences constructive et destructive lorsque $a=\sqrt{2}\lambda$, $a=\lambda$ et $a=\lambda/2$.

\sol{
$\delta = r-\sqrt{(x-a)^2+y^2)} \simeq ax/r \simeq a \cos(\theta)$. Constructif pour $\delta = p \lambda$ donne $\cos(\theta) = p \lambda/a$.
}


\item Déterminer l'expression exacte de la différence de marche $\delta(x)$ entre les deux ondes en un point de l'axe $Ox$. Déterminer le lieu des interférences constructive et destructive lorsque $a>\lambda$ et $a<\lambda$.

\sol{
$\delta = |x|-|x-a|=a$ si $x>a$, $2x-a$ si $0<x<a$ et $-a$ si $x<0$.
}

\end{quest}
\end{exo}

\begin{exo}[MPSI][1]{Quelques propriétés des ondes sonores}

On admet que les ondes sonores peuvent être décrites par un champ scalaire $s(x,t)$ qui vérifie l'équation d'onde
$
\frac{\partial^2 s}{\partial x^2} - \frac{1}{c^2}\frac{\partial^2 s}{\partial t^2} = 0
$
avec $x$ la position et $t$ le temps.


\begin{quest}

\item De quelle nature sont les ondes sonores ? Quelle grandeur pourraît représenter~$s$ ?

\sol{
mécanique, longitudinal. vitesse ou pression.
}

\item Déterminer la dimension de la constante $c$. Par analyse dimensionnelle, déterminer son expression dans l'air  en fonction de la masse molaire $M$, de la constante des gaz parfaits $R$ et de la température $T$. Quelle est la valeur de $c$ pour l'air ? l'eau ? un métal ?

\sol{
célérité en $m.s^{-1}$, $c=345\,$m$\p$s$^{-1}$ pour l'air, $c=1\,500\,$m$\p$s$^{-1}$ pour l'eau, $c=6\,000\,$m$\p$s$^{-1}$ pour un métal.
}

\item Montrer qu'une fonction $s(x,t)=f(x\pm ct)$ est solution de cette équation de propagation. Comment appelle-t-on ce type de solutions ?

\sol{
$$
\dfrac{\partial^2 s}{\partial x^2} - \dfrac{1}{c^2}\dfrac{\partial^2 s}{\partial t^2} = f''(x \pm vt)(1 - \dfrac{v^2}{c^2}) = 0
$$
donc soit $f''=0$ et $f$ affine, soit $v=c$ et $f$ quelconque. Onde progressive.
}

\item On cherche $f$ sous la forme d'une fonction sinusoïdale. Écrire alors la forme la plus générale de $s(x,t)$,
% en interprétant chaque terme. En illustrant éventuellement la définition par un schéma,
et définir la fréquence, la pulsation, la période, la phase, la longueur d'onde et le vecteur d'onde.

\sol{
$s(x,t) = A\cos[k(x-ct)] + B\sin[k(x-ct)] + C\cos[k(x+ct)] + D\sin[k(x+ct)]$. $k=2\pi/\lambda$ vecteur d'onde, $\omega = kc=2\pi \nu$ pulsation. Phase $\phi=kx-\omega t$.
}


\end{quest}

récepteur sont sur un même axe, séparés
d'une distance $d = 85\,$mm. Ces deux appareils
sont reliés à un oscilloscope. Les salves sont
visualisées à l'écran.
